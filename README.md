# Contact-Book
Description
This is a Contact Book application developed using C++.
It allows users to manage contacts by performing basic operations such as adding, viewing, searching, updating, and deleting contact details.

Features
Add new contacts
View all saved contacts
Search contact by name
Update contact details
Delete a contact
Simple console based interface

Technologies Used
Programming Language C++
Any standard C++ compiler like GCC MinGW or Turbo C++

How to Run
Step 1 Install a C++ compiler on your system
Step 2 Open terminal or command prompt
Step 3 Compile the program using
g++ contactBook.cpp -o contactBook
Step 4 Run the program using
contactBook

File Structure
contactBook.cpp contains the main source code
README.md contains project documentation
